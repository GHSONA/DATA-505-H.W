  filter(province=="Oregon" | province=="California" | province=="New York") %>%

  mutate(lprice=log(price)) %>% 
  select(lprice, points, cherry, province)

Multiple Regression

Linear Models

First run a linear regression model with log of price as the dependent variable and ‘points’ and ‘cherry’ as features (variables).

m1 <- lm(lprice ~ points + cherry, data=wine)
rmse_m1 <- sqrt(mean(residuals(m1)^2))
print(paste("RMSE for Model 1:", round(rmse_m1, 3)))
[1] "RMSE for Model 1: 0.469"

Explanataion:

TODO: lm(lprice ~ points + cherry, data=wine): Create a linear regression model where lprice is the dependent variable and points and cherry are the independent variables.

TODO: The RMSE of 0.469 indicates that our model’s predictions deviate from actual log prices by about 0.469 log units on average, or approximately 46.9% error in price predictions. This level of error is expected given the complexity of wine pricing factors.

Interaction Models

Add an interaction between ‘points’ and ‘cherry’.

m2 <- lm(lprice ~ points * cherry + province, data=wine)
rmse_m2 <- sqrt(mean(residuals(m2)^2))
print(paste("RMSE for Model 2:", round(rmse_m2, 3)))
[1] "RMSE for Model 2: 0.463"

TODO: lm(lprice ~ points cherry + province, data=wine): Create a linear regression model that includes interaction between points and cherry, as well as province as an independent variable.

TODO: The RMSE improved from 0.469 to 0.463 by adding the interaction term and province variable. This indicates that the more complex model better captures wine pricing patterns.

The Interaction Variable

The coefficient on the interaction term (points:cherry) means that the effect of having a “cherry” description on the wine’s price is influenced by the wine’s rating. For wines that mention “cherry”, each additional point in rating has a slightly greater impact on increasing the price compared to wines that do not mention “cherry”.

Applications

Determine which province (Oregon, California, or New York), does the ‘cherry’ feature in the data affect price most?

province_effect <- wine %>%
  group_by(province) %>%
  summarize(cherry_effect = mean(cherry))
print(province_effect)
